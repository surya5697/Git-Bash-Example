    Document Object Model

    Application programming interface

    Description

    The Document Object Model is a cross-platform and language-independent interface that treats an HTML or XML document as a tree structure wherein each node is an object representing a part of the document. The DOM represents a document with a logical tree. Wikipedia
    Abbreviation: DOM
    Base standards: WHATWG DOM Living Standard; W3C DOM4
    First published: October 1, 1998; 24 years ago
    Latest version: DOM4; November 19, 2015; 7 years ago
    Organization: World Wide Web Consortium, WHATWG
    Types
    The W3C DOM standard is separated into 3 different parts:
    • Core DOM - standard model for all document types.
    • XML DOM - standard model for XML documents.
    • HTML DOM - standard model for HTML documents.
